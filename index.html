Game Guess the Number

Браузер будет загадывать случайное четырехзначное число, а мы будем отгадывать. Попытки отгадать число будут идти через диалоговое окно — prompt. Браузер будет сообщать в ответ, больше или меньше загаданного наш ответ.

Task 1:

С помощью цикла while вывести все простые числа в промежутке от 0 до 100.

Task 2:

Организовать такой массив для хранения товаров в корзине

Task 3:

Организовать функцию countBasketPrice, которая будет считать стоимость корзины

Task 4:

* Вывести с помощью цикла for числа от 0 до 9, не используя тело цикла. Выглядеть это должно так: for(...){// здесь пусто}

Task 5:

* Нарисовать пирамиду с 20 рядами с помощью console.log, как показано на рисунке: x xx xxx xxxx xxxxx
